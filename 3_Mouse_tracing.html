Tracing mouse location



Behavorial Experiment

One of the major pitfall of bahvorial experiment is that it not only has a lot of external variables but more importantly they are extremely laborious and time consuming. Here I've demonstrated a quick script for mouse tracing in order to apply to wide variety of bahavorial assays. Instead of staring at the screen and recording the movement of mice, you can now can now go and enjoy Netflix instead.

Raw Video File

This is a recorded video of mice given a odor preference / avoidance test. In the left room, there is a no odor control whereas the right side of the room contains a sulfur bad smelling odor. The goal is to see if mouse has preference to either rooms.
However, in order for the computer to recognize and find the mouse location, we have to first convert the image to a binary input that's easier detectable by the computer.

Video Processing

The first step is to find the region of interest, this way we can limit the video down to smaller size with less error and noise. After series of image processing and threshold conversion, we're able to get a much cleaner video that can be easier detected by the software.

Here we can broadly observe that most chemical groups does not have an global impact in driving overall OR activation. However, certain groups of odor are robust enough to deviate, such as the thiazole and Ketones category.

Tracing and labeling video

Now that we have a nice and clean video outlining the location of the mouse. We can simply label the original video file using the thresholded location with better visual markers and traces to see mouse's trajectory path.

All of the following analysis and code can be found in my mouse tracing repository.
